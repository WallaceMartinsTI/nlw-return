## :lock: Servidor
Backend da aplicação Feedback Widget.

## ✨ Tecnologias

-   [ ] Express
-   [ ] Cors
-   [ ] Nodemailer
-   [ ] Prisma

## Executando o projeto

Utilize o **yarn** ou o **npm install** para instalar as dependências do projeto.
Em seguida, inicie o servidor.

**Em desenvolvimento**
```cl
npm run dev
```
**Em produção**
```cl
npm run start
```
